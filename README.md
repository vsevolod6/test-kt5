# Demo Web Shop Tests

Автоматизированные тесты сайта  
https://demowebshop.tricentis.com/

Используются:
- Python
- Selenium
- pytest
- webdriver-manager

---

# 1. Требования

Установленный **Python 3.10+**

Установка зависимостей

pip install selenium pytest webdriver-manager

Запуск тестов

python -m pytest -v
